Help



Welcome to Scolarly! Scolarly is designed to help you with your study related needs. If you require help in using Scolarly, please see below.





NOTES



All Notes


Notes can be opened by pressing this button on the home page.






Notes list is opened as shown below.






Add Note


To add a Note, press Add Note.






A Note can be created.






View Note


To view a Note, press the View button.






A Note will be displayed as shown.






Edit Note


To edit the Note, press the Edit button.






The Note can be edited.






Delete Note


To delete a Note, press the Delete button.






You will be asked to confirm if you want to delete the Note.




By pressing Confirm, the Note will be deleted.










ASSIGNMENTS



All Assignments


Assignments can be opened by pressing this button on the home page.






Assignments list is opened as shown below.








Add Assignment


To add an Assignment, press Add Assignment.






An Assignment can be added.






Mark Assignment as Done


To mark an Assignment as done, press the respective done checkbox. The Assignment will be added to the Done list.












The Assignment is in the Done list.






View Assignment


To view an Assignment, press the View button.






An Assignment will be displayed as shown.






Edit Assignment


To edit the Assignment, press the Edit button.






The Assignment can be edited.






Delete Assignment

You will be asked to confirm if you want to delete the Assignment.




By pressing Confirm, the Assignment will be deleted.
